# Claude Models Overview Documentation

## Current Models

**Claude Sonnet 4.5** is Anthropic's recommended starting point, offering "the best balance of intelligence, speed, and cost for most use cases, with exceptional performance in coding and agentic tasks."

### Key Specifications

All current models support:
- Text and image input, text output
- Multilingual capabilities and vision processing
- Extended thinking feature
- Priority tier access
- Maximum output of 64K tokens (Opus: 32K)

**Knowledge Cutoffs:**
- Sonnet 4.5: January 2025 (reliable), July 2025 (training data)
- Haiku 4.5: February 2025 (reliable), July 2025 (training data)
- Opus 4.1: January 2025 (reliable), March 2025 (training data)

## Model Capabilities

Claude 4 models excel in reasoning, coding, multilingual tasks, long-context handling, and image processing. The documentation notes these models produce "engaging responses" ideal for applications requiring rich, human-like interactions.
