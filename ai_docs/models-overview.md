# Claude Models Overview

## Current Models Available

### Claude Sonnet 4.5
- **Description**: Our smartest model for complex agents and coding
- **Pricing**: $3/input million tokens, $15/output million tokens
- **Context Window**: 200K tokens (1M beta available)
- **Max Output**: 64K tokens
- **Knowledge Cutoff**: January 2025
- **Recommendation**: Recommended for most use cases; offers the best balance of intelligence, speed, and cost

### Claude Haiku 4.5
- **Description**: Our fastest model with near-frontier intelligence
- **Pricing**: $1/input million tokens, $5/output million tokens
- **Context Window**: 200K tokens
- **Max Output**: 64K tokens
- **Knowledge Cutoff**: February 2025

### Claude Opus 4.1
- **Description**: Exceptional model for specialized reasoning tasks
- **Pricing**: $15/input million tokens, $75/output million tokens
- **Context Window**: 200K tokens
- **Max Output**: 32K tokens
- **Knowledge Cutoff**: January 2025

## Key Capabilities

All current models support:
- Text and image input with text output
- Multilingual functionality
- Extended thinking capabilities
- Priority tier access
- Vision processing

Available platforms:
- Anthropic API
- AWS Bedrock
- Google Vertex AI

## Getting Started

The documentation recommends starting with **Claude Sonnet 4.5** for optimal balance of intelligence, speed, and cost. Users can:
- Test models via [claude.ai](https://claude.ai)
- Make API calls through the Anthropic console
- Explore capabilities through the interactive platform

## Legacy Models

Older Claude 3 series models remain available, but migration to current versions is encouraged for enhanced performance improvements.
