# Shared Assets

Centralized brand assets for the Ozean Licht Ecosystem. Contains logos, design elements, and illustrations for both **Kids Ascension** and **Ozean Licht** platforms.

## Quick Reference

| Need | Location | Format |
|------|----------|--------|
| Favicon | `./favicon.webp` | WEBP |
| OL Logo (web) | `ozean-licht-logo/00_With_Text/WEBP/` | WEBP |
| KA Logo | `kids-ascension-logo/KidsAscension.webp` | WEBP |
| UI Accents | `design-elements/` | PNG |
| Payment badges | `Klarna Badge.webp`, `Paypal Badge.webp` | WEBP |

## Brand Usage

- **Ozean Licht**: Turquoise theme, see `ozean-licht-logo/README.md` for KeyCode system
- **Kids Ascension**: Separate branding, use assets from `kids-ascension-logo/`
