# Tools - Infrastructure & Automation

Progressive disclosure tool system for the Ozean Licht Ecosystem. 54+ commands organized into 6 categories, reducing context usage by 85%.

## Quick Start

```bash
# Smart discovery - natural language to command
bash tools/what.sh "deploy application"
bash tools/what.sh "check system health"

# Browse all categories
bash tools/discover.sh

# Learn about a command
bash tools/learn.sh "deploy vs restart"

# Every command supports --explain
bash tools/deployment/deploy.sh --explain
```

## Six Tool Categories

| Category | Purpose | Commands |
|----------|---------|----------|
| `deployment/` | Coolify operations | deploy, status, logs, restart, health |
| `containers/` | Docker operations | ps, logs, restart, stats, exec |
| `monitoring/` | Health & metrics | health-all, resources, connectivity |
| `database/` | PostgreSQL ops | backup, restore, query, size |
| `git/` | Version control | status, commit, push, pull |
| `remote/` | SSH operations | exec, upload, download, test |
| `memory/` | Institutional memory | save, search, patterns, health |
